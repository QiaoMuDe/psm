## ✨ 核心特性

| 特性 | 说明 |
|:-----|:-----|
| 📝 **Prompt 管理** | 增删改查、分类筛选、标签搜索、JSON 导入导出、模板变量、置顶 |
| 🧩 **Skill 管理** | 文件系统存储 + SKILL.md 自动解析，ZIP 双格式导入导出 |
| 🌐 **AI 翻译** | 左右分栏布局、流式翻译、6 种常用语言、语言交换、动画交互 |
| 🤖 **AI 生成/优化** | 一句话描述流式生成提示词、AI 优化内容/名称/描述、模型列表获取、连接测试 |
| 🎨 **6 种主题** | light/dark/midnight/ocean/rose/lavender + auto，支持跟随系统 |
| 💾 **完整备份** | ZIP 格式全量备份，一键恢复，同名自动跳过 |
| ⌨️ **快捷键** | 全局快捷键 + 模块快捷键，操作更高效 |
| 🖥️ **跨平台** | 基于 Wails v2，单文件部署，零外部依赖 |

## 🚀 快速开始

### 下载安装

从 [Gitee Releases](https://gitee.com/MM-Q/psm/releases) 下载对应平台的可执行文件，直接运行即可。

### 从源码构建

```bash
git clone https://gitee.com/MM-Q/psm.git
cd psm

# 开发模式
wails dev

# 生产构建
wails build
```

构建完成后，可执行文件位于 `build/bin/psm.exe`。
