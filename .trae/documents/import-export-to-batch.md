# 导入/导出按钮迁移到批量管理栏

## 实施计划

### 一、目标
将提示词模块和技能模块的导入/导出按钮从工具栏移动到批量管理栏中，同时为技能模块在工具栏添加"新增"按钮。

### 二、修改文件列表

#### 1. `frontend/js/views/prompts.js`
**当前状态分析：**
- 导入/导出按钮位置：工具栏（第59-70行）
- 批量管理栏位置：第82-94行（当前只有批量删除和退出按钮）
- 导入事件绑定：第546行
- 导出事件绑定：第559行

**修改步骤：**
1. **从工具栏移除导入/导出按钮**
   - 删除第59-70行的导入和导出按钮HTML
   - 确保工具栏保持简洁，只保留"批量管理"和"新增"按钮

2. **在批量管理栏添加导入/导出按钮**
   - 在批量删除按钮后面添加导入和导出按钮
   - 按钮样式需要与批量管理栏的整体设计保持一致

3. **更新事件绑定**
   - 确保导入/导出事件绑定代码能够正确找到新的按钮元素
   - 测试按钮在批量管理模式下的显示/隐藏逻辑

4. **调整CSS样式**
   - 确保批量管理栏有足够的宽度容纳新增的按钮
   - 按钮间距和布局需要合理
